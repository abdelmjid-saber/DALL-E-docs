## To install the script, follow the steps below:
1. Extract the downloaded zip file on your PC.
2. Upload the installer.zip file to your server in any directory you want to run the application.
3. Extract the zip file in that directory. 
4. Open client/src/page/home.jsx file.
5. Set http://localhost:8000/api/v1/post to Your Server Domain like http://dall-e.tk/api/v1/post
6. Open client/src/page/CreatePost.jsx file.
7. Set http://localhost:8000/api/v1/dalle to Your Server Domain like http://dall-e.tk/api/v1/dalle
8. Set http://localhost:8000/api/v1/post to Your Server Domain like http://dall-e.tk/api/v1/post
9. Open server/.env file.
10. Set OPENAI_API_KEY: Your OPENAI Connection API, get an API key by visiting [Link](https://beta.openai.com/account/api-keys "title")
11. Set MONGODB_URL: Your MongoDB Connection String URI. [Demo](https://www.mongodb.com/docs/manual/reference/connection-string "title")
12. Set CLOUDINARY_CLOUD_NAME: Your Cloudinary Connection String Name. [Demo](https://cloudinary.com/documentation/how_to_integrate_cloudinary#1_create_your_account_and_set_up_your_product_environment "title")
13. Set CLOUDINARY_API_KEY: Your Cloudinary Connection API Key. [Demo](https://cloudinary.com/documentation/how_to_integrate_cloudinary#1_create_your_account_and_set_up_your_product_environment "title")
14. Set CLOUDINARY_API_SECRET: Your Cloudinary Connection API Secret. [Demo](https://cloudinary.com/documentation/how_to_integrate_cloudinary#1_create_your_account_and_set_up_your_product_environment "title")

## Installing Module
* Run ```npm install``` in your terminal application client directory.
* Run ```npm install``` in your terminal application server directory.

## Build Project
* Run ```npm run build``` in your terminal application client directory.

## Ready To Upload
You can now upload on any hosting with the required requirements\
We have explained [how to Deploy to Render and Vercel hosting](https://dall-e-docs.vercel.app/deploy-for-render&vercel "title")

## Run On Local Host
* Run ```npm run preview``` in your terminal application client directory.
* Run ```npm run start``` in your terminal application server directory.
